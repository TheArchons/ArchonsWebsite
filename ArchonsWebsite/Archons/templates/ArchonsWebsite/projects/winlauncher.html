WinLauncher

This was a short project to create a program launcher in the terminal. My use case was to be able to quickly run games, but it may be used for any program. Written in C++ with the pdcurses library.

I learned how to write terminal applications with C++ and create UIs with pdcurses. I also learned how to interact with the Win32 API.
